#  2023 교내 캡스톤 디자인 졸업작품 커뮤니티 강의실 예약 웹 사이트
📣 주제

본교 강의실과 좌석을 선택해 이용자가 필요로하는 좌석을 시간대 별로 예약 기능 
게시판에 글을 작성해 등록과 글 검색 가능, 작성된 게시글에 댓글 작성 기능

# 백엔드
> Spring boot

> H2

# 프론트 엔드
> html

> css

> thymeleaf


# 📣 기능 목록
#### 회원 도메인
> 회원가입

> 로그인 
#### 게시판 
> 글 등록

> 글 수정/삭제

> 파일 올리기

#### 게시판 댓글
> 댓글 작성자

> 내용

> 댓글 작성한 시간
 
#### 강의실 예약 
> 해당 강의실의 시간/ 분 등록 예약

> 등록된 예약 조회
